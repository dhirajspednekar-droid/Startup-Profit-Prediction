# 📊 Startup Profit Prediction Using Multiple Linear Regression

## 📌 Project Overview

This project analyzes startup expenditure data to identify the factors affecting business profit using Multiple Linear Regression. The project combines SQL, Python, and Power BI to extract, analyze, predict, and visualize business insights.

---

## 🎯 Objective

- Analyze startup expenditure data
- Identify factors affecting profit
- Build a Multiple Linear Regression model
- Predict future profit
- Create an interactive Power BI dashboard

---

## 🛠️ Tools & Technologies

- Python
- SQL (MySQL)
- Power BI
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook

---

## 📂 Dataset

The dataset contains **50 startup records** with the following attributes:

- R&D Spend
- Administration
- Marketing Spend
- State
- Profit

---

## 📈 Project Workflow

1. Data Extraction using SQL
2. Data Cleaning & Preprocessing
3. Exploratory Data Analysis (EDA)
4. Multiple Linear Regression Model
5. Profit Prediction
6. Dashboard Development using Power BI

---

## 📊 Model Performance

- Model: Multiple Linear Regression
- R² Score: **0.8987**
- Prediction Accuracy: **89.87%**

---

## 💡 Key Insights

- R&D Spend has the highest positive impact on profit.
- Marketing Spend also positively influences profit.
- Administration Spend has minimal impact.
- Higher investment in R&D leads to higher profitability.

---

## 📌 Recommendations

- Increase investment in R&D.
- Continue effective marketing strategies.
- Optimize administrative expenses.
- Focus on innovation-driven growth.
